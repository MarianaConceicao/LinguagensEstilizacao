# LinguagensEstilizacao
Aprendizado sobre estilização Web

Projeto desenvolvido para a disciplina de Programação Front-End, com o objetivo de aplicar conceitos de desenvolvimento web utilizando HTML e CSS.

## Descrição

Este projeto consiste na criação de um site institucional inspirado na temática aeroespacial, com base em uma interface moderna e responsiva.

A aplicação apresenta diferentes seções, incluindo área principal (hero), apresentação de conteúdo, exibição de carreiras e rodapé com formulário, simulando uma página corporativa.

## Funcionalidades

- Layout responsivo
- Menu de navegação
- Seção principal com destaque (hero)
- Seções informativas com imagens e texto
- Cards de carreiras
- Rodapé com formulário de contato

## Tecnologias utilizadas

- HTML5
- CSS3

## Estrutura do projeto

- `index.html` → Estrutura da página
- `estilo.css` → Estilização e layout
- `imagens/` → Recursos visuais utilizados no site

## Conceitos aplicados

- Estruturação semântica com HTML
- Flexbox e Grid Layout
- Responsividade com media queries
- Organização de estilos com variáveis CSS
- Boas práticas de separação entre estrutura e estilo

## Como executar

1. Baixar ou clonar o repositório
2. Abrir o arquivo `index.html` em um navegador
